# 2025-10-25 搭建 Android 工程骨架

## 变更概要
- 新增可运行的最小 Android 工程骨架（Kotlin + Jetpack Compose + Material3 + Hilt + Room/KSP + WorkManager + CameraX）。
- 配置 Hilt 使用 kapt 编译器（Room 继续使用 KSP）。
- 新增 Application、MainActivity、Compose 主题与基础资源；修正 Manifest 主题与组件命名为全限定名。

## 新增文件
- app/src/main/AndroidManifest.xml
- app/src/main/java/com/lessup/medledger/MedLedgerApp.kt
- app/src/main/java/com/lessup/medledger/MainActivity.kt
- app/src/main/java/com/lessup/medledger/ui/theme/Theme.kt
- app/src/main/res/values/strings.xml
- app/src/main/res/values/themes.xml
- app/proguard-rules.pro

## 修改文件
- app/build.gradle.kts
  - 添加 `org.jetbrains.kotlin.kapt` 插件
  - Hilt 编译器由 KSP 切换至 kapt
  - 添加 `com.google.android.material:material:1.12.0`
  - 添加 `kapt { correctErrorTypes = true }`

## 说明
- 主题采用 `Theme.MedLedger`（继承 `Theme.Material3.DayNight.NoActionBar`），Compose 主题由 `AppTheme` 承载（支持动态色）。
- 当前未包含 Gradle Wrapper；建议在 Android Studio 打开工程直接 Sync，或本地安装 Gradle 后执行 `gradle wrapper` 生成 Wrapper。
